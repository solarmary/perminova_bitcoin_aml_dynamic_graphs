# Parser Data

Эта папка предназначена для локально сгенерированных данных парсера.

- 
aw/ — raw JSON Bitcoin-блоков.
- interim/ — промежуточные Parquet-таблицы.
- processed/ — итоговые Elliptic-like CSV/Parquet.

Данные не коммитятся в GitHub. Для воспроизведения используйте команды из parser/README.md.
