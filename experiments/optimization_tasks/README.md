# Optimization Tasks Experiments

Раздел содержит материалы эксперимента по пяти оптимизационным задачам на синтетическом динамическом графе.

## Задачи

- Minimum Spanning Tree / Minimum Spanning Forest.
- Shortest Path.
- Maximum Cut.
- Minimum Vertex Cover.
- Top-k Anomalous Temporal Subgraph Discovery.

## Содержимое

- `notebooks/5experiments-last-try_v2.ipynb` — канонический ноутбук эксперимента.
- `results/` — небольшие итоговые CSV с метриками по задачам.
- `figures/` — визуализации качества, времени работы и устойчивости методов.
- `THEORETICAL_EXPERIMENT_PLAN.md` — исходный план постановки эксперимента.

Крупные таблицы `edges_df`, `nodes_df`, `delta_df` не перенесены, так как они воспроизводятся из ноутбука и не подходят для обычного GitHub-репозитория.
